# Skill System - 能力可视化系统

## 产品需求文档 (PRD) v1.0

---

## 1. 产品概述

### 1.1 产品定位
Skill System 是一个**游戏化个人能力可视化平台**，将用户的成长过程转化为 RPG 游戏式的"系统面板"体验。用户可以像游戏角色一样查看自己的能力属性、技能树、等级进度，并获得 AI 助手的全程陪伴指导。

### 1.2 核心概念
- **系统流体验**：模拟小说中的"系统"概念，用户是主角，拥有可视化的属性面板
- **打怪升级**：将学习、工作、生活中的成长转化为经验值和等级提升
- **AI 系统助手**：24/7 在线的 AI 助手，负责制定计划、提供建议、鼓励提醒

### 1.3 目标用户
- 自我提升爱好者
- 终身学习者
- 游戏化生活追求者
- 需要可视化管理个人成长的人群

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 能力属性面板 (Character Stats Panel)
**功能描述**：展示用户的核心能力属性，类似 RPG 角色属性界面

**属性分类**：

| 属性类别 | 子属性 | 说明 |
|---------|--------|------|
| **身体素质** | 力量、敏捷、耐力、健康 | 运动、健身相关能力 |
| **智力属性** | 智商、记忆力、逻辑思维、创造力 | 认知能力 |
| **社交能力** | 魅力、领导力、沟通能力、情商 | 人际关系能力 |
| **专业技能** | 编程、设计、写作、语言等 | 可自定义的职业技能 |
| **心理状态** | 专注力、意志力、幸福感、压力值 | 心理健康指标 |
| **生活技能** | 时间管理、财务管理、组织能力 | 实用生活能力 |

**可视化方案**：
1. **雷达图**：展示多维度能力分布
2. **进度条**：各属性的当前等级和经验值
3. **数值面板**：具体数值 + 等级标识
4. **历史曲线**：能力成长的时间轴趋势
5. **对比视图**：与上周/上月/去年的自己对比

#### 2.1.2 技能树系统 (Skill Tree)
**功能描述**：可视化的技能学习路径，类似游戏技能树

**设计要素**：
- 树状结构展示技能依赖关系
- 已解锁/可解锁/未解锁状态区分
- 点击技能节点查看详情
- 学习路径推荐

**示例技能树**：
```
编程基础
├── 前端开发
│   ├── HTML/CSS
│   ├── JavaScript
│   ├── React/Vue
│   └── 全栈开发
├── 后端开发
│   ├── Node.js
│   ├── Python
│   ├── 数据库
│   └── 系统架构
└── 移动开发
    ├── iOS
    ├── Android
    └── 跨平台
```

#### 2.1.3 等级与经验系统 (Level & XP)
**功能描述**：量化成长进度，提供即时反馈

**等级体系**：
- **新手期** (Lv.1-10)：白色边框
- **进阶期** (Lv.11-30)：绿色边框
- **高手期** (Lv.31-60)：蓝色边框
- **专家期** (Lv.61-99)：紫色边框
- **大师期** (Lv.100+)：金色边框 + 特效

**经验获取方式**：
- 完成任务
- 学习新技能
- 坚持习惯
- 达成里程碑
- 获得成就

**升级特效**：
- 升级动画（光效、粒子效果）
- 全屏通知
- 属性提升展示
- AI 助手祝贺语音/文字

#### 2.1.4 任务与成就系统 (Quests & Achievements)
**功能描述**：将目标拆解为可执行的任务

**任务类型**：
- **主线任务**：重要目标（如"学会 React"）
- **支线任务**：辅助提升（如"读一本技术书"）
- **日常任务**：习惯养成（如"每天运动 30 分钟"）
- **限时任务**：挑战活动

**成就系统**：
- 连续打卡成就
- 技能突破成就
- 等级里程碑成就
- 特殊事件成就

#### 2.1.5 AI 系统助手 (AI Assistant)
**功能描述**：贯穿全程的 AI  companion

**助手功能**：
1. **计划制定**
   - 根据目标生成学习路径
   - 拆解任务到可执行步骤
   - 推荐最优学习资源

2. **进度跟踪**
   - 监控用户活跃度
   - 提醒未完成任务
   - 庆祝升级和成就

3. **智能建议**
   - 瓶颈期诊断与突破建议
   - 学习方法优化
   - 能力短板补强方案

4. **情感支持**
   - 鼓励性话语
   - 挫折时的安慰
   - 成就时的庆祝

5. **可视化生成**
   - 自动分析数据生成最佳图表
   - 推荐适合的数据展示方式
   - 生成进度报告

**交互方式**：
- 侧边栏聊天界面
- 语音/文字输入
- 主动推送通知

| 部署 | Vercel | 前端托管 |

### 4.2 数据库模型

```prisma
// 用户模型
model User {
  id            String    @id @default(cuid())
  email         String    @unique
  name          String
  level         Int       @default(1)
  totalXP       Int       @default(0)
  attributes    Attribute[]
  skills        Skill[]
  tasks         Task[]
  achievements  Achievement[]
  aiChats       AIChat[]
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
}

// 属性模型
model Attribute {
  id        String   @id @default(cuid())
  userId    String
  user      User     @relation(fields: [userId], references: [id])
  name      String   // 力量、智力等
  category  String   // 身体素质、智力属性等
  level     Int      @default(1)
  currentXP Int      @default(0)
  maxXP     Int      @default(100)
  history   Json?    // 历史数据
}

// 技能模型
model Skill {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  name        String
  category    String
  level       Int      @default(0)
  parentId    String?
  parent      Skill?   @relation("SkillTree", fields: [parentId], references: [id])
  children    Skill[]  @relation("SkillTree")
  unlocked    Boolean  @default(false)
  progress    Float    @default(0)
}

// 任务模型
model Task {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  title       String
  description String?
  type        String   // daily, main, side, limited
  xpReward    Int
  completed   Boolean  @default(false)
  dueDate     DateTime?
  completedAt DateTime?
}

// AI 聊天记录
model AIChat {
  id        String   @id @default(cuid())
  userId    String
  user      User     @relation(fields: [userId], references: [id])
  role      String   // user, assistant, system
  content   String
  createdAt DateTime @default(now())
}
```

## 5. 用户体验设计

### 5.1 首次使用流程
1. **欢迎动画**：系统激活特效
2. **能力评估**：AI 引导用户完成初始能力评估
3. **目标设定**：选择想要提升的方向
4. **系统初始化**：生成初始属性面板和推荐任务
5. **新手引导**：介绍各功能区域

### 5.2 日常使用流程
1. 查看今日任务和属性状态
2. 完成任务获得经验
3. 与 AI 助手交流获取建议
4. 查看成长进度和图表
5. 规划下一阶段目标

### 5.3 特殊时刻
- **升级时刻**：全屏动画 + AI 祝贺
- **成就解锁**：徽章展示 + 分享功能
- **连续打卡**：火焰特效 +  streak 展示
- **里程碑**：年度/季度总结报告

## 6. AI 助手设计

### 6.1 助手人格设定
- **名称**：System（系统）
- **性格**：专业、鼓励、略带幽默
- **说话风格**：简洁有力，偶尔使用游戏术语

|-----|---------|

| 瓶颈期 | 诊断分析 + 突破建议 |

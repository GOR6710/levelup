# LevelUp - 游戏化个人能力可视化平台

## 🎮 项目简介

LevelUp 是一个将个人成长游戏化的可视化平台。像 RPG 游戏一样管理你的技能、属性和任务，让自我提升变得有趣且可视化。

## ✨ 核心功能

- **属性面板** - 六维能力雷达图，直观展示你的能力分布
- **等级系统** - 通过完成任务获得经验值，升级解锁新能力
- **任务系统** - 主线/支线/日常任务，游戏化目标管理
- **技能树** - 可视化技能学习路径
- **AI 助手** - 24/7 在线的系统助手，提供建议和陪伴

## 🛠️ 技术栈

- Next.js 14 + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion 动画
- Recharts 数据可视化
- Prisma + PostgreSQL

## 🚀 快速开始

```bash
npm install
npm run dev
```
